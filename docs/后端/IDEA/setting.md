### 展开 Java 文件中的 import \* 🐝

1. 打开 `Preferences:`

![1676771110604](./image/setting/1676771110604.png)

2. 进入 Editor > Code Style > java， 打开 `import`Tab, 复选框勾选第一项 `Use single class import`, 并修改底部标注的部分为一个较大的数值，如 99. 最后保存修改。

![1676771234537](./image/setting/1676771234537.png)
